# Excel Sales Project

## 📊 Project Overview
Sales Dashboard built using Microsoft Excel with Pivot Tables and Data Analysis.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs

## 📈 Dashboard Features
- Total Sales: ₹51,39,340
- Total Units Sold: 1805
- Total Profit: ₹18,07,250
- Average Sales: ₹1,97,667
- Region wise Sales
- Product wise Sales
- Sales Person Performance
